Talks

Delivering a multi-sub-brand design system for e-commerce

Config 2024, San Francisco U.S.A.

In 2023, Zalando revamped its design system to create varied "Themes" for different propositions like Fashion and Beauty. This talk explores their journey in developing this capability and launching the first theme.

Gonzalo Vasquez, Leonie Proske, Sheryl Bagga

Navigating complex system updates

Schema 2022, London U.K.

Zalando's design system team supports 50+ teams and 140k weekly component inserts. This talk uses one example to show how they use branching, contribution processes, and custom tools to manage complex system updates in close collaboration with embedded teams.

Gonzalo Vasquez, Leonie Proske

Introducción al pensamiento en sistemas (Systems thinking)

Garaje Inspirational Fest 2025, Spain (online)

This talk explains why systems thinking is critical for designers and tech professionals. I explored how understanding systems—including flows, stocks, and feedback loops—helps us see problems comprehensively and intervene effectively for better outcomes.
